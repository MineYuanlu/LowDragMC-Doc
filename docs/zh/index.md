# LowDragMC

!!! info inline end "项目简介"
    ![项目简介](../assets/profile.png){ width="105%"}

LowDragMC 并非一个活跃的团队，而是一个汇集了各种有趣模组项目的集合。

这些项目的唯一维护者是 **[KilaBash](https://github.com/Yefancy) :)**  

---

:simple-discord: [加入我们的 Discord](https://discord.com/invite/sDdf2yD9bh)  

:simple-github: [GitHub 仓库](https://github.com/Low-Drag-MC)  

本质上，这里更像是我本人和许多优秀开发者**独立项目的一个枢纽**，而非一个结构严密的团队。

我们致力于开发**富有创意的模组**，为用户提供**自定义能力**，并特别关注**技术和渲染方面**。

### **我们的模组列表**:
1. [**LDLib**](ldlib/index.md)
2. [**LDLib2 (1.21+)**](ldlib2/index.md)
3. [**Multiblocked & Multiblocked2**](multiblocked2/index.md)
4. [**Photon & Photon2**](photon/index.md)
5. **Shimmer**
